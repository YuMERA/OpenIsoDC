# OpenIsoDC

**OpenIsoDC** is an open-source engineering project focused on the design, development and documentation of an isolated **5V to 5V DC/DC converter**.

Unlike many hardware projects that only publish the final schematic, OpenIsoDC documents the complete engineering process—from the first prototype to the final design.

## Project Goals

- Design an isolated 5V → 5V DC/DC converter
- Minimum output current: 500 mA
- Target output current: 1 A
- High efficiency
- Low ripple and noise
- Fully documented development process
- Open-source hardware and documentation

## Current Status

🚧 Project started.

The first development phase focuses on evaluating several converter topologies before selecting the final architecture.

## Repository Structure

```
docs/            Documentation
hardware/        Schematics, PCB and transformer designs
calculations/    Design calculations
oscilloscope/    Measurements and waveforms
images/          Images and diagrams
tools/           Helper scripts and utilities
```

## Development Philosophy

Every design decision should be supported by measurements.

Every prototype should be documented.

Every failure is valuable engineering data.
