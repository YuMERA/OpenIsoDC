# Project Requirements Specification (PRS)

## Project

**Name:** OpenIsoDC

**Version:** 0.1

**Status:** Draft

---

# Mission

Develop an open-source isolated 5V to 5V DC/DC converter while documenting the complete engineering process.

The project aims to produce not only a working converter, but also a complete engineering reference including design decisions, measurements, calculations and prototype evolution.

---

# Primary Goal

Design an isolated USB powered DC/DC converter with the following minimum requirements:

| Parameter | Value |
|-----------|-------|
| Input Voltage | 5 VDC |
| Output Voltage | 5 VDC (isolated) |
| Minimum Output Current | 500 mA |
| Target Output Current | 1 A |

---

# Development Philosophy

- Measurements before conclusions.
- Every prototype must be documented.
- Every design decision must have a reason.
- Failed experiments are valuable engineering data.
- The complete development process will be public.
